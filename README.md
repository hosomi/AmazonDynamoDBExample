# AmazonDynamoDBExample


## DynamoDB local

:link: [DynamoDB ローカル (ダウンロード可能バージョン) のセットアップ - Amazon DynamoDB](https://docs.aws.amazon.com/ja_jp/amazondynamodb/latest/developerguide/DynamoDBLocal.html)  
:link: [JDK Builds from Oracle](http://jdk.java.net/)  

```cmd
java -jar DynamoDBLocal.jar -sharedDb
```

__http://localhost:8000/__  


## DynamoDB local front

:link: [dynamodb-admin - npm](https://www.npmjs.com/package/dynamodb-admin)  
:link: [Node.js](https://nodejs.org/ja/)  


```cmd
dynamodb-admin
```

__http://localhost:8001/__
